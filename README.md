
# BIG BRAIN

This is a school project. It's a memory game with Overwatch theme. The objective of this app is learning how to manage android view's lifecycle. This app follows the architecture of MVVM.  


## Tech Stack

**Language:** Kotlin

**SDK Version:** 32

**Tags:** ViewModels.
